#TestEngine
Test Engine is a Desktop Based application intended for teachers and students. Test Engine is developed using swing, core java and postgreSQL.   Teachers and students can register and login. Teachers will create test in excel sheet and upload the excel. Application is then able to read excel and store test in database. Students can give their respective tests after login.
